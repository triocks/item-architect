# PF2E: Triocks Item Architect

**Architect**: Item Architect is a procedural forging hub for Pathfinder 2e. Designed for Game Masters who need precision and flexibility, this module allows you to build custom items from the ground up using a modular "Rule Element" engine. Whether you are crafting a simple +1 longsword or a complex artifact, you can stack detailed rules, runes, and materials to define exactly how an item behaves in the PF2e system.

**Procedural Forging**: Generate base items that adhere strictly to PF2e math, levels, and rarity whitelists.

**Modular Rule Stacking:** Layer multiple properties, runes, and custom logic onto a single base item.

**Drafting UI**: A dedicated workspace to preview and "Architect" your items before committing them to your world.

**System Integration**: Built specifically for the PF2e system on Foundry VTT, utilizing native rule elements and fundamental rune progression.

**v13 Ready**: Fully optimized for the latest Foundry VTT environments using modern Vanilla JS.

ðŸš€ **Installation**
Method 1: Manifest URL (Recommended)
Open the Foundry VTT Configuration and Setup screen.

Go to the Add-on Modules tab.

Click Install Module.

Paste the following URL into the Manifest URL field: https://github.com/triocks/item-architect/releases/latest/download/module.json

Click Install.

Method 2: Manual Installation
Download the module.zip file from the Latest Release.

Extract the contents into your Foundry Data folder: Data/modules/item-architect.

Restart Foundry VTT.

ðŸ“– How to Use
**Launch the Architect**: Go to the Items Tab in the right-hand sidebar. Click the Architect button in the header.

**Select Base**: Choose the item category (Weapon, Armor, etc.) and the desired level.

**Draft Your Item:** Use the Drafting UI to add materials, runes, and special rule elements.

**Forge**: Once satisfied with the configuration, click Forge Item to generate the finished actor-ready item in your world.  Items will appear in the 'Item Architect' folder.
